# Online Examination System

A C++ console-based Online Examination System that allows students to attempt multiple-choice questions, receive instant evaluation, and store examination results. The project demonstrates object-oriented programming, file handling, and basic exam management functionality.

## Features

- Student registration using name and roll number
- Multiple-choice question-based examination
- Random question selection from question bank
- Automatic answer evaluation
- Score calculation
- Instant feedback for correct and incorrect answers
- Result storage using file handling
- Object-oriented design using classes

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- STL (Vectors, Algorithms)

## Concepts Implemented

- Classes and Objects
- Encapsulation
- Constructors
- Vector Data Structure

## How to Run

### Using g++

1. Clone the repository:

```bash
git clone https://github.com/NavyaYadav25/online-examination-system.git
```

2. Navigate to the project folder:

```bash
cd online-examination-system
```

3. Compile the program:

```bash
g++ main.cpp -o exam
```

4. Run the application:

```bash
./exam
```

## Working Flow

1. Student enters name and roll number.
2. System randomly selects questions from the question bank.
3. Student answers MCQ questions.
4. Answers are evaluated automatically.
5. Final score is displayed.
6. Result is saved in `results.txt`.
